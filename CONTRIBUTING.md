# Contributing to Fortytwode

First off, thank you for considering contributing! <3

The project was created for the 42 students and alumni of Hive Helsinki, but we also welcome contributions from non-42 students at Hive or with anyone with a 42 network account ✨

## Bugs & Features

### To report a bug, or suggest a feature:

Search our [issues](https://github.com/EvAvKein/Fortytwode/issues?q=is%3Aissue) to see if the topic has an existing issue post:
- If you find an existing issue that matches what you were thinking of, read through the post and comments. If the problem hasn't been resolved yet or you have a relevant question, feel free comment there!
- If no existing issue matches what you want to report/contribute, create a new issue using the "Standard issue" template.

### To contribute code:

We try to label the most approachable issue tickets as "good first issue", but contributions to other ones are also very welcome!

If there's an issue that you want to help fix, feel free to fork the repo and submit a pull request that tags the relevant issue in the description. Make sure to update tests and documentation as relevant.

## Anything else

If you have any meta questions or ideas, or for anything else, either:
- Post a "Blank issue" template.

OR

- Message [me](https://github.com/EvAvKein) (Eve, the maintainer) through one of the profiles/addresses on my page.
